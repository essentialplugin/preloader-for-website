Preloader for Website : A loading screen add-on for your WordPress website.

== Description ==
Preloader  gives your site an animated loading screen.

Check [Features and Demo](https://www.essentialplugin.com/wordpress-plugin/preloader-for-website/?utm_source=WP&utm_medium=Preloader&utm_campaign=Read-Me) for additional information.

It's a great and lightweight way to add spice to any site. Simply activate the plugin and enable the Loader from "Preloader – WPOS" menu tab.

It works everywhere. It’s been tested on desktop browsers along with a variety of iOS, Android and Windows devices.

== Installation ==

1. Upload the 'preloader-for-website' folder to the '/wp-content/plugins/' directory.
2. Activate the "preloader-for-website" list plugin through the 'Plugins' menu in WordPress.
3. Go to Preloader - WPOS then select enable preloader and select your spinner.
4. Now you can see preloader on your website.
